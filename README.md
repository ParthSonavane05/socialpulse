📌 PresencePulse — Stay Truly Present

A privacy-first behavioral intelligence mobile application that detects unconscious phone usage (phubbing) and helps users stay mindful during real-world interactions.

⸻

🚀 Overview

In a world dominated by smartphones, people often lose real-life presence without realizing it.
PresencePulse is designed to solve this problem by acting as a digital presence coach — helping users become aware of their phone habits and encouraging mindful engagement.

Built during a 24-hour hackathon at LNMIIT Jaipur, this app combines behavioral science, mobile sensing, and modern UI/UX design.

⸻

🎯 Problem

“Phubbing” — ignoring people around you to use your phone — leads to:
	•	Reduced relationship quality
	•	Lower attention span
	•	Increased loneliness
	•	Poor communication

Existing solutions fail because they:
	•	Only show analytics
	•	Depend on self-control
	•	Lack real-time awareness

⸻

💡 Solution

PresencePulse introduces a context-aware behavioral intelligence system that:
	•	Detects real-world interaction scenarios
	•	Learns user behavior patterns
	•	Identifies abnormal phone usage spikes
	•	Provides real-time, gentle nudges
	•	Encourages mindful digital habits

🏗️ Architecture

The app follows a privacy-first, on-device architecture:
	•	📱 Flutter-based mobile app
	•	🧠 Local behavioral intelligence engine
	•	💾 On-device storage (Hive / SQLite)
	•	📡 Sensor-based data collection
	•	☁️ Optional Firebase for demo & scalability
  
 🧪 How It Works
	1.	Collects behavioral signals (unlock frequency, notifications, motion)
	2.	Builds a personal usage baseline
	3.	Detects abnormal usage spikes
	4.	Identifies possible social interaction context
	5.	Triggers real-time nudges
	6.	Stores insights for reflection

⸻

🔐 Privacy First
	•	All processing is done on-device
	•	No personal data is stored externally
	•	Optional cloud features are anonymized
